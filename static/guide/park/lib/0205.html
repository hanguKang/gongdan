수납 및 변경은 어떻게 하나요?

성북구 도시관리공단 주차사업팀은 보다 빠르고 편리한 민원서비스를 위해 최선을 다하겠습니다.

수납안내

  • ① 현장방문 : 홈페이지 주차사업팀소개=> 위치확인후 방문 수납 (현금, 카드수납 가능)
  • ② OCR 고지서 : 가까운 금융기관 또는 인터넷 지로납부사이트 (http://www.giro.or.kr) 접속 후 납부
  • ③ 카드납부 : 차주와 명의가 동일 하지 않아도 결제가능( 사무실 내방, 인터넷 결제)
  • ④ 가상계좌 : 고지서에 나온 개인별 가상계좌로 결제가능 (국민, 농협, 신한, 우리, 우체국 사용 가능)

결제시 문의사항은 주차사업팀 (914-2008)으로 연락주시어 입금 확인 바랍니다.

변경신청 안내

모든 변경신청은 주차팀 홈페이지 로그인 후 마이페이지에서 변경 가능합니다.

인터넷 사용이 어려우신분은 내방 혹은 아래 양식을 첨부하여 팩스를 보내주시면 즉시 변경 가능합니다.

거주자우선주차증 재발생 신청서(양식) 다운로드

  • ① 차량변경 : 차량번호 변경시 변경신청서작성 후 변경된 차량등록증사본을 제출
  • ② 할인내역변경: 장애인, 국가유공자, 다둥이, 경차 변경시 변경신청서작성 후 관련서류사본을 제출
  • ③ 구획변경 : 주차구획 변경시 해당동 담당자 확인 후 변경신청서 작성
  • ④ 주차방법변경 : 주차방법 변경시(주간, 야간, 전일) 해당동 담당자 확인(914-2008 유선가능) 후 변경신청서 작성

모든 변경신청시 수납방법(요금)의 변동사항이 있을 경우 추가요금 또는 환불이 발생할 수 있습니다.
